사용 방법

  1. 1. 이미지 업로드 페이지에서 학습할 이미지와 해당 이미지의 라벨을 입력합니다.
  2. 2. 각 카테고리별로 충분한 수의 이미지(최소 10개 이상 권장)를 업로드합니다.
  3. 3. 모델 학습 페이지에서 학습 매개변수를 설정하고 학습을 시작합니다.
  4. 4. 학습이 완료되면 이미지 테스트 페이지에서 새로운 이미지를 업로드하여 분류 결과를 확인합니다.
